    从SELECT开始

    • 先来一个内联结的使用例子。使用联结就要记得加上联结条件，不然的话会得出笛卡尔积
    • 组合查询，将多条SELECT语句组合成为一个结果集，就是使用UNION。有两种情况需要用到组合查询：一个查询中从不同的表返回结构数据；对一个表执行多次查询，按一个查询返回结果。这个玩意写起来就比较复杂，，但是在使用多个表查询的时候，UNION处理会更加简单。UNION会自动取消掉重复的行，如果有需要，可以使用UNION ALL

    插入的几种方式

    1. 插入完整的行
    2. 插入行的一部分
    3. 插入某些查询结果
    • 好的方式插入，这种方式可以插入全部信息或者部分信息，只要你指定好了列名。
    • 差的方式插入，明显就是没有指定列名，有一定的危险。一定要全部项都去给定一个值。
    • 插入检索出来的数据,就是指定的列的数据，是通过其他表中检索出来的
    • 从一个表复制到另外一个表

    简介：

    • 删除表中的所有行。没有写WHERE语句

    • 删除表中的特定行

    • 简单用法

    如果想要删除表中所有的行，不要使用DELETE语句，使用TRUNCATE TABLE 语句

    操作表语句，CREATE TABLE，ALERT TABLE，DROP TABLE

    CREATE TABLE这个，指定主键，指定类型，指定是否可以为NULL，指定默认值。NULL不是空字符串，不用混为一谈。那是什么都没有，是一个无效的值。而空字符串是一个有效的值。

    ALERT TABLE更新表结构
    可以增加列或者删除列等。不过，要是真要这样去改，还不如新建一个新的表，把需要的数据从旧的表中复制过来。删旧表，更新旧表名到新表。

    DROP TABLE 这个就是删除表
